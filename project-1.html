Quiz Game

Using React, I created a multiple choice quiz that draws on an online database of questions.

Project Overview

Having completed a React fundamentals course with the online code-learning platform Scrimba, I took on their self-led solo challenge, to create a fun quiz app. Given a Figma mockup of a user interface to replicate and a list of functional requirements to fulfil, it was a demanding task that really tested my grasp of core React and JS, and flexed my CSS and HTML skills. It was as satisfying to create as it is to play and win. Some key challenges along the way included:

  • Meeting the requirement to load in the quiz questions and answers through a third-party API (a database of open-source trivia questions) – solved using JavaScript’s fetch API, Promises, async functions and React’s useEffect hook.
  • Cleaning the source quiz data, to ensure HTML entities were properly displayed.
  • Restructuring the data into a suitable format, including deciding and adding custom properties to track which answer a user had selected.
  • Managing state within the app, such as tracking the user’s selected answers, checking them at the end of the round and, naturally, starting a fresh game.
  • Writing all HTML and CSS from scratch, to create a user interface that matched the Figma mockup. This included carefully tuning details like hover states for the answer buttons to ensure a clean glitch-free user experience. Plus, of course, a couple of easter eggs, including a confetti cannon when you get all the answers right!
